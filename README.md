## 考后总结
* 这次的经济比重比我预想的大好多，资料弄得稍有点偏。

## 已完成：
* 教材要点
  * 很多大段大段的话总结起来只有一句。
  * “X是A、B、C的Y”和“做到X，就是要A、B、C、D”式的话是选择题喜欢考的。
  * 一共10页，建议每页3分钟。
* 四中全会
  * 不管你看不看得进去，务必用10分钟的时间认真阅读[会议公报](http://www.xinhuanet.com/politics/2019-10/31/c_1125178024.htm)。
